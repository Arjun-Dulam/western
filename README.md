# **My WezTerm Config**

My personal [WezTerm](https://wezterm.org) configuration for macOS.

## Install WezTerm

```sh
brew install --cask wezterm
```

## Setup

Clone the repo into your WezTerm config directory:

```sh
git clone https://github.com/adulam/wezterm-config ~/.config/wezterm
```
